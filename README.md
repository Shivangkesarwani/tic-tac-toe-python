#  Tic Tac Toe Game

A GUI-based Tic Tac Toe game developed using Python and Tkinter.

## Features

* Two-player gameplay
* Player name input
* Turn management
* Winner detection
* Draw detection
* Score tracking
* Winning combination highlighting
* Restart game functionality

## Technologies Used

* Python
* Tkinter
